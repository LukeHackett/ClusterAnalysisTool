3D Map Plug-in

In order to use the 3D map, an additional plug-in is required. The Google Earth Browser plug-in can be installed from this link.

Please note that Google Earth is not required to be installed in order for this plug-in to work. This plug-in will work regardless of whether Google Earth is installed or not. The plug-in does not get installed when Google Earth is installed, so if Google Earth is installed upon your machine, the plug-in will need to be installed.
